# Tari explorer

## No client-side JavaScript block explorer (tari-text-explorer)

### Dependencies

- `npm install`

### Development (nodemon reload)

- `npm run dev`

### Start server

- `npm start`

## Useful environment variables

- PORT:4000 'Port of service'
- BASE_NODE_GRPC_URL:localhost:18142 'Base node GRPC URL'
